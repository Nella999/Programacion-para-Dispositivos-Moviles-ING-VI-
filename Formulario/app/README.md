# Formulario Mejorado con componentes Compose

## Descripción
Aplicación desarrollada en Android Studio usando Kotlin y Jetpack Compose.

Consiste en un formulario interactivo con validaciones y componentes modernos de Compose.

## Funcionalidades

- Campo Nombre
- Campo Edad
- Campo Correo Electrónico
- RadioButton para género
- Checkbox para aceptar términos
- Switch usuario activo/inactivo
- Slider para nivel
- Botón Registrar inteligente
- Botón Limpiar
- Mensaje de registro exitoso

## Validaciones

- Nombre no vacío
- Edad solo números
- Correo contiene @
- Términos aceptados

## Tecnologías usadas

- Kotlin
- Android Studio
- Jetpack Compose
